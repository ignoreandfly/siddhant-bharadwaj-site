VS Code Shortcut Keys That You Must Know

According to a 2016 developers survey of Stack Overflow, Visual Studio Code ranked 13th among the top popular development tools, with only 7% of the 47,000 respondents using it.
Five years later, in the 2021 developer survey, it was ranked 1 with 70% of the 82,000 respondents using it. With so many developers using Visual Studio Code, let’s dive into some of the shortcut keys that you should know to make the most out of Visual Studio Code.

Shortcut keys:

  • Alt + Z: Toggle Wrap - It takes the text running off the screen and wraps it within the window.
  • Ctrl + O: Open file - As the name suggests it opens a file
  • Ctrl + ~: Show/Hide Terminal - It shows or hide the terminal on the left side of the screen
  • Ctrl + ‘: Show/Hide Sidebar - It shows or hide the sidebar at the bottom of the screen
  • Ctrl+P followed by : Jump to the line - It allows you to jump to a particular line.
  • Ctrl+P followed by @: Jump to the symbol- It allows you to jump to a particular example say ‘body’ for example.
Note: If you’re a Mac user, you need to press cmd instead of Ctrl to use these keys.
