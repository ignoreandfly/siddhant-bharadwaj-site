Types of Input in HTML

Any program can basically be broken down into three chunks i.e. input, processing and output in that order.

Input which is the first part of what constituents a program is taken using input tag in HTML, in this blog post we will take about various types of inputs that you can use.

HOW TO USE DIFFERENT TYPES OF INPUT?

WHAT ARE THE VARIOUS KINDS OF INPUT?

  • Text - It is ideal when the user has to enter text
  • Password- In this type of input, the password field is shown as asterisks
  • Submit - It defines a button for submitting the form
  • Reset- It is a button that will reset all the values to default values
  • Radio- It is useful while designing quizzes, it allows users to only choose one out of several options
  • Checkbox- It allows users to choose one or more than one option out of several options
  • Date- It is used for the input field which contains a date
  • File-This allows users to submit a file present either on their computer or somewhere else
  • Number- It defines a number input field
You can read about some other types of input that are rarely used here: https://www.w3schools.com/html/html_form_input_types.asp
